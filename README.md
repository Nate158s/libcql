libcql
======

Implementation of the [Cassandra binary protocol](https://github.com/apache/cassandra/blob/trunk/doc/native_protocol.spec) in C++

### Current Features
- Completely Asynchronous
- Exception Safe
- Minimal Memory Copies
- SSL Support
- Ad-hoc Queries
- Prepared Statements
